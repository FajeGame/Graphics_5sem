# Растеризация отрезка и окружности

Простая программа для демонстрации алгоритмов растеризации на квадратной сетке.

## Описание

Программа создает квадратную сетку с размером ячейки 2-3 мм и реализует:
- Алгоритм Брезенхема для растеризации отрезка
- Алгоритм Брезенхема для растеризации окружности

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python rasterization.py
```

## Как это работает

1. **Создание сетки**: Программа создает квадратную сетку с размером ячейки примерно 2.5 мм
2. **Растеризация отрезка**: Используется алгоритм Брезенхема для определения, какие ячейки сетки должны быть закрашены при отрисовке отрезка
3. **Растеризация окружности**: Также используется алгоритм Брезенхема, который рисует окружность, используя симметрию относительно осей координат

Результат сохраняется в файл `rasterization_result.png` и отображается на экране.
