Простая консольная программа, превращающая фото/рисунок в схему для вышивки крестиком с палитрой DMC.

## Установка
```bash
pip install Pillow numpy scikit-learn
```

## Использование
```bash
py cross_stitch.py
```
Далее введите:
1) Название изображения (например: photo.jpg)
2) Количество цветов (по умолчанию: 20, допустимо: 1–50)
3) Размер схемы в крестиках (по умолчанию: 100, допустимо: 10–500)

## Результат
В папке с изображением появятся:
- <имя>_schema.png — схема: цветные клетки + символы, счётные линии каждые 10 клеток
- <имя>_legend.png — легенда: образец цвета, символ, номер DMC

## Палитра DMC
- Источник палитры: [sventana.com/dmc](https://sventana.com/dmc/)
- Формат файла `dmc_colors.json` (JSON), пример записи:
```json
{
  "dmc": "310",
  "name": "Black",
  "rgb": [0, 0, 0]
}
```

## Как это работает
- Масштабирование изображения с сохранением пропорций
- Извлечение цветов пикселей (svg‑pooling: 1 пиксель → 1 клетка; усреднение даёт ресайз)
- Кластеризация K‑means до N цветов
- Поиск ближайших цветов из палитры DMC (евклидово расстояние в RGB)
- Отрисовка схемы (клетки + символы) и легенды; счётные линии через 10 клеток

## Зависимости
- Pillow — работа с изображениями
- NumPy — векторные вычисления (RGB, расстояния)
- scikit‑learn — кластеризация K‑means
